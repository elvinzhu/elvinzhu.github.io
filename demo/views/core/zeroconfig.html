zero_config

页面控制器路径，可传入“default”使用控制器基类的基本功能
页面数据， 支持各种数据类型，会直接传递给controller.run 方法
类型：anytype
使用场景：页面需要输出一些需要js处理的数据
requirejs 配置项
类型：object
是否自动弹出加载controller 的loading框。
类型：boolean
使用场景：页面定制了自己的loading框（仅仅重写loading样式的除外）
是否自动隐藏加载controller的loading框。
类型：boolean
使用场景：页面一开始就需要弹出自己的loading框
controler使用default时需要扩展到默认控制器上的方法（注：可以在这里注册run方法）
类型：object
使用场景：页面需要少量逻辑，没有必要新添加一个js文件。
例子： 
<button event-src="doLittle">实例</button>
<script>
var zero_config = {
    controller: 'default',
    data: { },
    actions:{
        doLittle: function(e){
            // this.$('#id')
        }
    }
};
</script>
用于区别编译和非编译环境
类型：sring
使用场景：本地开发
