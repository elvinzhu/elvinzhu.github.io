数据选择器

new DataPicker(options)

options.title{string}弹窗标题。默认“请选择”
options.data{object|array}显示数据。支持的格式如下:
// 1. 数组
[1,2,3,4];
// 2. 对象数组
[{
    key:'', name:''
}];
 // 3. 纯对象
{
    key1: 'value1',
    key2: 'value2',
}

on(type, callback)

type{string}事件类型。暂时只有 change, 在用户选择某项之后触发。会给回调传递{key: "4", name: "军人证"}形式的用户选择的数据
注： BaseController中提供了一个pick_item（传送门）事件处理器来简化datapicker的使用
